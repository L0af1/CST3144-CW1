## Main Functions 
- list of tutors loaded from the backend  
- Made a filter having:
  - Subject  
  - Maximum price  
  - Minimum rating  
  - Location  
- **Search-as-you-type** (name, subject, or location)
- View the tutor profiles
- Add tutors to a booking cart
- Submit a booking form
- space reduction on successful bookings
- Reactive UI using Vue’s Composition API
- Took my time making a styled interface using a silver & purple professional theme

## Technologies used 
- **Vue.js 3** (script setup)
- **Vite** (build tool)
- **Vue Router**
- **Reactive Store** (custom store using Vue’s `reactive`)
- **Fetch API** for communication with backend
- **GitHub Pages** for deployment

### **Install dependencies**
 npm install

 npm run build (for development purposes only)

 npm run dev 

 then it should run at local host: http://localhost:5173

 - **Gets deployed on docs folder root since i had to use github pages**
